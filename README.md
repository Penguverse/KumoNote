# KumoNote

**KumoNote** is a lightweight, Electron-based note-taking application by **Penguverse Studios**. Designed for speed, simplicity, and reliability, it provides a modern desktop note experience with rich features.

---

## Features

- **Dark & Light Themes:** Switch between dark and light modes.
- **Custom Title Bar:** Minimalist window controls for Windows.
- **Rich Text Formatting:** Bold, italic, underline, strikethrough, code blocks, lists, headings, and font size adjustments.
- **File Management:** New, Open, Save, Save As, and Export as PDF.
- **Auto-Save:** Keeps your work safe using local storage or direct file saving.
- **Undo/Redo & History:** Easy editing with up to 100 history steps.
- **Fast & Lightweight:** Single-page Electron app optimized for performance.
